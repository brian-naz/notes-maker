# Notes Maker

A Google Keep Clone using Material UI 2.

To get started:
Demo

### Setup:

Clone the repo:
```bash
https://github.com/brian-naz/notes-maker.git
```
Go to the directory and simply type:
```bash
npm install
```
To run:
```bash
npm start
```
